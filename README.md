# Exploratory Data Analysis (EDA)

## Objective
Perform Exploratory Data Analysis (EDA) on the Financial Sales dataset using Python to identify trends, relationships, and patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Financial Sales Dataset

## Analysis Performed
- Dataset Overview
- Missing Value Check
- Summary Statistics
- Value Counts
- Histogram
- Boxplot
- Scatter Plot
- Correlation Heatmap
- Pairplot

- Gross Sales, Sales, COGS, and Profit are strongly correlated.
